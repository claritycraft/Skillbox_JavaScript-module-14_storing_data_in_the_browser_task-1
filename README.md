### En:
#### Task 2
What You Need to Do
Improve the functionality of the application you worked on in the video “Working with JSON and LocalStorage” from the module “Storing Data in the Browser”.
Implement additional features in the project and bring the web interface in line with the provided layout.

List of Additional Features
1. Deleting a movie
Implement a function that allows the user to delete a movie from the list.
Each movie row in the table should have a “Delete” button.
When clicked, the movie should be removed from both the HTML table and localStorage.

2. Editing movie information
Implement a function that allows the user to edit movie details.
Each movie row in the table should have an “Edit” button.
When clicked, the form should be automatically filled with the movie’s information, which can then be updated.
Updating the form should change the data in both the HTML table and localStorage.
3. Sorting the movie list
Implement a function that allows the user to sort movies by title, genre, and release year.
Add a dropdown menu to select the sorting criteria.
4. Form validation for adding a movie
Implement form validation to ensure that the title, genre, and release year fields are filled before submission.
If validation fails, display an appropriate error message.

### Ru:
#### Задача 2
Что нужно сделать
Доработайте функциональность приложения, с которым вы работали в видео «Практика работы с JSON и LocalStorage» в модуле «Хранение данных в браузере». Реализуйте в проекте дополнительную функциональность и приведите вид веб-интерфейса к макету.

Список дополнительной функциональности

Удаление фильма.
Реализуйте функцию, которая позволит пользователю удалить фильм из списка. В таблице рядом с каждым фильмом должна быть кнопка «Удалить». По нажатию на эту кнопку фильм должен быть удалён как из HTML-таблицы, так и из localStorage.
Редактирование информации о фильме.
Реализуйте функцию, позволяющую редактировать детали фильма. В таблице рядом с каждым фильмом должна быть кнопка «Редактировать». По нажатию на эту кнопку форма должна быть автоматически заполнена информацией о фильме, которую затем можно обновить. Обновление формы должно изменить данные как в HTML-таблице, так и в localStorage.
Сортировка списка фильмов.
Реализуйте функцию, которая позволит пользователю сортировать фильмы по названию, жанру и году выпуска. Добавьте выпадающий список для выбора критерия сортировки.
Валидация формы для добавления фильма.
Реализуйте валидацию формы, чтобы перед её отправкой убедиться, что поля с названием, жанром и годом выпуска фильма заполнены. Если валидация не пройдена, необходимо показать соответствующее сообщение об ошибке.
